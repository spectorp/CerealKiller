Hi, I'm Perry Spector, and I built Cereal Killer in three weeks as an Insight Data Science Fellow. Several people in my life have severe food allergies, and it's tedious for them to manually sift through packaged foods at the grocery store to find safe products. I used computer vision methods to detect and identify cereal boxes in an image and then communicate to the user which ones are safe.


Under the hood:

Data: Training an object detection model requires annotated images. Rather than manually identifying cereal boxes in images, I generated thousands of artificial images of cereal boxes superimposed on images of grocery store shelves, like the one below. Although I only used a few images of each cereal box, I increased the diversity of box images by blurring, modifying the color, and applying projective transformations. I compiled cereal and allergy information in a MySQL database.

Algorithm: I trained the YOLO v3 model to detect the locations of cereal boxes in images using transfer learning. For each detected box, I use the Cloud Vision API to identify text. Next I use careful string matching to classify each box, and then query a database for allergen information.

Backend: The project is coded in Python and leverages numpy, PIL, pandas, Keras, BeautifulSoup, and sqlalchemy.

Frontend: Built using Flask and matplotlib. Deployed on AWS.

The source code is available on GitHub.
